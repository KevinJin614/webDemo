Hibernate API
Configuration	用于解析hibernate.cfg.xml和Xxx.hbm.xml,并创建SessionFactory对象
SessionFactory	用于创建Session的工厂，很消耗资源，线程安全 单例
Session		持久化管理器，程序和数据库之间的会话，线程不安全 1.synchronized 2.每个人分配新的Session对象
Transaction	事务管理器
